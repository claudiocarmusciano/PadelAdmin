De la idea al producto: así está avanzando PadelAdmin 🎾

Hace algunas semanas arrancamos el desarrollo de PadelAdmin, una plataforma de gestión integral de torneos de pádel. Hoy quiero compartir el estado actual del proyecto y lo que ya está funcionando.

¿Qué es PadelAdmin?

Una app web pensada para organizadores de torneos que necesitan gestionar jugadores, armar zonas, generar fixtures automáticos con restricciones horarias, registrar resultados y llevar el bracket eliminatorio.

🛠️ Stack tecnológico

Backend

  • Spring Boot 4 / Java 21
  • PostgreSQL
  • API REST

Frontend

  • React + Vite + TypeScript
  • Tailwind CSS v4
  • shadcn/ui (dark theme)
  • React Query + Axios
